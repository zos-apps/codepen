## About

Online code editor for HTML, CSS, and JavaScript with live preview.

## Features

- Live preview
- HTML/CSS/JS editors
- Console output
- Save pens

## Installation

### From App Store
Open the **App Store** in zOS and search for "CodePen".

### Via npm
```bash
npm install @zos-apps/codepen
```

## Usage

- Write code in editors
- See live preview
- Console shows JS output



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
