CodePen

Online code editor for HTML, CSS, and JavaScript with live preview.

GitHub npm

Features

✓ Live preview
✓ HTML/CSS/JS editors
✓ Console output
✓ Save pens

Installation

From App Store

Open the App Store in zOS and search for "CodePen".

Via npm

npm install @zos-apps/codepen

How to Use

  • Write code in editors
  • See live preview
  • Console shows JS output

Development

# Install dependencies
npm install

# Start development
npm run dev

# Build for production
npm run build
